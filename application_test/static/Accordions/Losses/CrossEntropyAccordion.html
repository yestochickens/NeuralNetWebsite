Cross-Entropy is a loss commonly used in binary classification. It must be used after a function that finds the probability of each outcome, which is usually achieved through the Softmax activation function. Cross-entropy measures the differences in probability distributions, significantly penalizing confident wrong predictions, making it the usual go-to for any model with binary classification.
