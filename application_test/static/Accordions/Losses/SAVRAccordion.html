SAVR, short for Sum of the Absolute Value Residuals, is similar to SSR; however, the differences between the true label and predicted label are run through an absolute value function rather than squared. This allows errors to still add up, but doesn't give more weight to larger errors.

Predicted \[\mathbf{\hat{Y}}\] Expected \[\mathbf{Y}\]

Predicted \[\begin{bmatrix} \widehat{y_0} \\ \vdots \\ \widehat{y_{n_y}} \end{bmatrix}\] Expected \[\begin{bmatrix} y_0 \\ \vdots \\ y_{n_y} \end{bmatrix}\]

Equation \[J \left( \mathbf{\hat{Y}}, \mathbf{Y} \right) = \sum_{i=1}^{n_y} \left| y_i - \widehat{y_i} \right|\]

Equation \[J \left( \begin{bmatrix} \widehat{y_0} \\ \vdots \\ \widehat{y_{n_y}} \end{bmatrix}, \begin{bmatrix} y_0 \\ \vdots \\ y_{n_y} \end{bmatrix} \right) = \sum_{i=1}^{n_y} \left| y_i - \widehat{y_i} \right|\]
